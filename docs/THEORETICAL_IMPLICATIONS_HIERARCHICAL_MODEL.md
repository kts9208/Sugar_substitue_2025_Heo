# 계층적 ICLV 모델의 이론적 함의

**작성일**: 2025-11-11  
**목적**: 제안된 계층적 구조의 이론적 타당성 및 연구 기여도 분석

---

## 📊 모델 비교

### 현재 모델 (병렬 구조)
```
외생 LV → 구매의도 → 선택
  ├─ 건강관심도 ──┐
  ├─ 건강유익성 ──┤
  ├─ 가격수준 ────┼─→ 구매의도 ─→ 선택
  └─ 영양지식 ────┘
```

**특징**:
- 모든 외생 LV가 구매의도에 **직접 효과**
- 외생 LV 간 관계 없음 (독립)
- 단순하고 추정 안정적

### 제안 모델 (계층적 구조)
```
1차 LV → 2차 LV → 3차 LV → 선택 (조절효과)
건강관심도 → 건강유익성 → 구매의도 ─→ 선택
                                    ↗ (조절)
가격수준 ────────────────────────────┘
                                    ↗ (조절)
영양지식 ────────────────────────────┘
```

**특징**:
- **간접효과** 경로 존재
- **조절효과** 메커니즘 명확
- 이론적 정교함 증가

---

## 🎯 이론적 타당성 분석

### 1. 계층적 구조: 건강관심도 → 건강유익성 → 구매의도

#### **이론적 근거**

##### **Theory of Planned Behavior (TPB)**
- **태도 형성 과정**: 신념 → 태도 → 의도
- 건강관심도 (건강에 대한 일반적 관심) → 건강유익성 (특정 제품의 건강 혜택 인식) → 구매의도

##### **Health Belief Model (HBM)**
- **인지된 유익성 (Perceived Benefits)**: 건강 행동의 핵심 결정 요인
- 건강관심도가 높을수록 → 건강 유익성을 더 크게 인식 → 구매의도 증가

##### **Elaboration Likelihood Model (ELM)**
- **중심 경로 (Central Route)**: 건강관심도가 높은 사람은 건강 정보를 더 정교하게 처리
- 건강관심도 → 건강유익성 평가 → 구매의도

#### **실증적 근거**

**선행연구**:
1. Moorman & Matulich (1993): 건강 동기 → 건강 정보 처리 → 건강 행동
2. Dutta-Bergman (2004): 건강 관심 → 건강 신념 → 건강 행동
3. Kim & Chung (2011): 건강 관심도 → 기능성 식품 인지된 가치 → 구매의도

#### **✅ 결론: 이론적으로 매우 타당**

---

### 2. 조절효과: 가격수준과 영양지식이 구매의도 → 선택 관계 조절

#### **2.1 가격수준의 조절효과**

##### **이론적 근거**

**Price Sensitivity Theory**:
- 가격 민감도가 높을수록 구매의도와 실제 구매 간 괴리 증가
- **조절 방향**: 부적 조절 (λ₂ < 0 예상)
  - 가격수준 인식이 높을수록 (비싸다고 느낄수록) 구매의도가 실제 선택으로 전환되는 정도 감소

**Willingness-to-Pay (WTP) Theory**:
- 인지된 가격이 WTP를 초과하면 구매의도가 있어도 구매하지 않음

##### **실증적 근거**

**선행연구**:
1. Zeithaml (1988): 인지된 가격 → 구매 의도-행동 간 관계 조절
2. Dodds et al. (1991): 가격 인식이 품질-구매 관계 조절
3. Lichtenstein et al. (1993): 가격 의식이 태도-행동 관계 조절

##### **✅ 결론: 이론적으로 타당 (부적 조절 예상)**

---

#### **2.2 영양지식의 조절효과**

##### **이론적 근거**

**Knowledge-Attitude-Behavior (KAB) Model**:
- 지식이 태도와 행동 간 관계를 강화
- **조절 방향**: 정적 조절 (λ₃ > 0 예상)
  - 영양지식이 높을수록 구매의도가 실제 선택으로 전환되는 정도 증가

**Self-Efficacy Theory**:
- 영양지식이 높을수록 건강한 선택에 대한 자기효능감 증가
- 의도를 행동으로 옮기는 능력 향상

##### **실증적 근거**

**선행연구**:
1. Parmenter & Wardle (1999): 영양지식 → 식품 선택 행동
2. Wardle et al. (2000): 영양지식이 태도-행동 관계 강화
3. Dickson-Spillmann et al. (2011): 영양지식이 의도-행동 간극 감소

##### **✅ 결론: 이론적으로 타당 (정적 조절 예상)**

---

### 3. 사회인구학적 변수 효과 제거

#### **현재 모델**: 사회인구학적 변수 → 구매의도 (직접 효과)

#### **제안 모델**: 사회인구학적 변수 효과 제거

#### **이론적 검토**

##### **제거 찬성 의견**:
1. **매개 효과 가정**: 사회인구학적 변수는 건강관심도를 통해 간접적으로만 영향
2. **모델 간결성**: 불필요한 경로 제거로 해석 용이
3. **잠재변수 중심**: ICLV의 핵심은 잠재변수 효과

##### **제거 반대 의견**:
1. **직접 효과 가능성**: 연령, 소득이 구매의도에 직접 영향 가능
2. **통제 변수 역할**: 사회인구학적 변수는 통제 변수로 포함이 일반적
3. **모델 적합도**: 직접 효과 제거 시 적합도 저하 가능

#### **⚠️ 권장사항**:

**대안 1: 완전 제거** (제안대로)
- 이론적 근거가 명확한 경우
- 모델 간결성 우선

**대안 2: 1차 LV에 효과 추가** (MIMIC 모델)
```
사회인구학적 변수 → 건강관심도 → 건강유익성 → 구매의도
```
- 더 정교한 모델
- 사회인구학적 변수의 간접효과 분석 가능

**대안 3: 선택모델에 직접 효과**
```
V = β·X + λ·PI + δ·Sociodem
```
- 통제 변수로 포함
- 잠재변수 효과와 독립적

#### **✅ 결론: 이론적 근거에 따라 결정 (대안 2 권장)**

## 📈 연구 기여도

### 1. 이론적 기여

#### **1.1 계층적 메커니즘 규명**
- 건강관심도 → 건강유익성 → 구매의도 경로 실증
- 간접효과 분석 가능
- 이론적 정교함 증가

#### **1.2 조절효과 메커니즘 규명**
- 가격수준의 부적 조절효과 (장벽 역할)
- 영양지식의 정적 조절효과 (촉진 역할)
- 의도-행동 간극 (Intention-Behavior Gap) 설명

### 2. 방법론적 기여

#### **2.1 ICLV 모델 확장**
- 계층적 구조 + 조절효과 통합
- 기존 ICLV 모델의 한계 극복

#### **2.2 복잡한 관계 모델링**
- 직접효과, 간접효과, 조절효과 동시 추정
- 더 현실적인 소비자 의사결정 과정 반영

### 3. 실무적 기여

#### **3.1 마케팅 전략 수립**
- **건강관심도 증진**: 건강유익성 인식 향상을 통해 구매의도 증가
- **가격 전략**: 가격 장벽 완화 방안 (할인, 프로모션)
- **교육 프로그램**: 영양지식 향상으로 의도-행동 전환율 증가

#### **3.2 세분화 전략**
- **고가격 민감 집단**: 가격 할인 집중
- **저영양지식 집단**: 교육 프로그램 제공
- **고건강관심 집단**: 건강 혜택 강조

## 🔍 예상 가설

### H1: 계층적 구조
- **H1a**: 건강관심도는 건강유익성에 정적 영향 (γ₁ > 0)
- **H1b**: 건강유익성은 구매의도에 정적 영향 (γ₂ > 0)
- **H1c**: 건강관심도는 건강유익성을 통해 구매의도에 간접 영향 (γ₁ × γ₂ > 0)

### H2: 조절효과
- **H2a**: 인지된 가격수준은 구매의도와 선택 간 관계를 부적으로 조절 (λ₂ < 0)
  - 가격수준이 높을수록 구매의도가 실제 선택으로 전환되는 정도 감소
- **H2b**: 영양지식은 구매의도와 선택 간 관계를 정적으로 조절 (λ₃ > 0)
  - 영양지식이 높을수록 구매의도가 실제 선택으로 전환되는 정도 증가

### H3: 간접효과
- **H3**: 건강관심도는 건강유익성과 구매의도를 순차적으로 매개하여 선택에 영향

## 📊 예상 효과 크기

### 직접효과
```
건강관심도 → 건강유익성: γ₁ ≈ 0.5 ~ 0.7 (중간-강)
건강유익성 → 구매의도: γ₂ ≈ 0.6 ~ 0.8 (강)
구매의도 → 선택: λ₁ ≈ 1.0 ~ 1.5 (강)
```

### 간접효과
```
건강관심도 → 건강유익성 → 구매의도: γ₁ × γ₂ ≈ 0.3 ~ 0.56 (중간)
건강관심도 → 건강유익성 → 구매의도 → 선택: γ₁ × γ₂ × λ₁ ≈ 0.3 ~ 0.84 (중간-강)
```

### 조절효과
```
가격수준 조절: λ₂ ≈ -0.2 ~ -0.4 (약-중간, 부적)
영양지식 조절: λ₃ ≈ 0.2 ~ 0.4 (약-중간, 정적)
```

## ✅ 최종 권장사항

### 1. 계층적 구조
**✅ 강력 권장**: 이론적으로 매우 타당하며 연구 기여도 높음

### 2. 조절효과
**✅ 강력 권장**: 의도-행동 간극 설명에 중요한 메커니즘

### 3. 사회인구학적 변수
**⚠️ 조건부 권장**: 
- **우선**: 1차 LV (건강관심도)에 효과 추가 (MIMIC 모델)
- **차선**: 완전 제거 (제안대로)
- **최소**: 선택모델에 통제 변수로 포함

### 4. 구현 순서
1. **Phase 1**: 계층적 구조만 구현 (조절효과 없이)
2. **Phase 2**: 조절효과 추가
3. **Phase 3**: 사회인구학적 변수 경로 재검토
