## 1. 모델 구조 개요

### **ICLV 모델의 3가지 구성요소**

```
┌─────────────────────────────────────────────────────────────┐
│                     ICLV 모델 구조                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  [측정모델]                                                  │
│  건강관심도 (LV) → 지표 변수 (q6-q11)                        │
│  파라미터: ζ (요인적재량), τ (임계값)                         │
│                                                              │
│  [구조모델]                                                  │
│  사회인구학적 변수 (X) → 건강관심도 (LV)                     │
│  파라미터: γ (회귀계수)                                       │
│                                                              │
│  [선택모델]                                                  │
│  속성 (price, health_label) + 건강관심도 (LV) → 선택         │
│  파라미터: β (속성 계수), λ (잠재변수 계수)                   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 2. 측정모델 파라미터 (ζ, τ)

### **2.1 요인적재량 (ζ - Zeta)**

**의미**: 잠재변수(건강관심도)가 각 지표 변수에 미치는 영향

**모델**:
```
y*ᵢⱼ = ζⱼ × LVᵢ + εᵢⱼ
```

여기서:
- `y*ᵢⱼ`: 개인 i의 지표 j에 대한 잠재 응답
- `ζⱼ`: 지표 j의 요인적재량
- `LVᵢ`: 개인 i의 건강관심도 (잠재변수)
- `εᵢⱼ`: 오차항

### **2.2 임계값 (τ - Tau)**

**의미**: 순서형 프로빗 모델의 범주 경계값

**모델**:
```
y_ij = k  if  τ_(j,k-1) < y*_ij ≤ τ_(j,k)
```

여기서:
- `y_ij`: 관측된 응답 (1, 2, 3, 4, 5)
- `y*_ij`: 잠재 응답
- `τ_(j,k)`: 지표 j의 k번째 임계값

---

#### **임계값 구조 (5점 척도)**

**범주 정의**:
```
범주 1 (매우 그렇지 않다):  y* ≤ τ_1
범주 2 (그렇지 않다):       τ_1 < y* ≤ τ_2
범주 3 (보통이다):          τ_2 < y* ≤ τ_3
범주 4 (그렇다):            τ_3 < y* ≤ τ_4
범주 5 (매우 그렇다):       τ_4 < y*
```

---

#### **τ_1,1 ~ τ_1,4 (q6의 임계값)**

| Coefficient | Estimate | 의미 |
|-------------|----------|------|
| **τ_1,1** | -3.875 | 범주 1과 2의 경계 |
| **τ_1,2** | -2.891 | 범주 2와 3의 경계 |
| **τ_1,3** | -0.804 | 범주 3과 4의 경계 |
| **τ_1,4** | 1.596 | 범주 4와 5의 경계 |

**해석**:
- τ_1,1 = -3.875: 잠재 응답이 -3.875 이하이면 "매우 그렇지 않다" 선택
- τ_1,2 = -2.891: 잠재 응답이 -3.875 ~ -2.891이면 "그렇지 않다" 선택
- τ_1,3 = -0.804: 잠재 응답이 -2.891 ~ -0.804이면 "보통이다" 선택
- τ_1,4 = 1.596: 잠재 응답이 -0.804 ~ 1.596이면 "그렇다" 선택
- 잠재 응답이 1.596 초과이면 "매우 그렇다" 선택

**유의성**:
- τ_1,1, τ_1,2: p < 0.01 → 매우 유의함
- τ_1,3, τ_1,4: p > 0.10 → 비유의함

## 3. 구조모델 파라미터 (γ - Gamma)

**모델**:
```
LVᵢ = γ₁ × age_stdᵢ + γ₂ × genderᵢ + γ₃ × income_stdᵢ + ηᵢ
```

여기서:
- `LVᵢ`: 개인 i의 건강관심도
- `γ`: 회귀계수
- `ηᵢ`: 오차항 (표준정규분포)

## 4. 선택모델 파라미터 (β, λ)

**의미**: 선택 속성과 건강관심도가 선택에 미치는 영향

**모델**:
```
V = β_Intercept + β_price × price + β_health_label × health_label + λ × LV
P(선택=1) = Φ(V)
```

여기서:
- `V`: 효용 (Utility)
- `Φ`: 표준정규 누적분포함수
- `LV`: 건강관심도 (잠재변수)

   - 건강 라벨의 효과성 의문

3. **건강관심도의 역할 없음**:
   - 건강관심도가 높아도 선택에 영향 없음
   - 건강관심도와 실제 선택 행동의 괴리

4. **사회인구학적 변수의 역할 없음**:
   - 연령, 성별, 소득이 건강관심도를 설명하지 못함
   - 건강관심도는 개인 특성과 무관
